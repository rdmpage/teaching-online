This app takes a tree description and displays a lineage through time plot.

Step 1: Enter tree

The tree must be an ultrametric tree, that is have branch lengths, and every tip of the tree must be the same distance from the root. Paste in a tree description below and click Show.

A good source of ultrametric trees is Time tree
